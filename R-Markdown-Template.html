Section 3

This is going to make a plot!

Option + command + i will create an R markdown code chunk (ie three backticks {R}).

plot_df =
  tibble(
    x = rnorm(n = 1000),
    y = 1 + 2 * x + rnorm(n = 1000)
  )

ggplot(plot_df, aes(x = x, y=y)) + geom_point()

For R Markdown, you need to load all the packages you need before running R code, For the above code for example, you need to reload the library(tidyverse) so it can run tibble and ggplot. Load your packages in a separate codechunk at the very beginning of the doc!
